# AI Job Portal

An AI-powered Job Portal web application built with Django that connects job seekers and recruiters with smart resume analysis and job matching features.

## Features

- User Registration & Login
- Recruiter Dashboard
- Job Seeker Dashboard
- Post Jobs
- Apply for Jobs
- Resume Upload (PDF)
- AI Resume Analysis
- ATS Score Checking
- Skill Extraction from Resume
- Job Matching based on Resume Skills

## Tech Stack

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite
- Groq API
- PyMuPDF (PDF parsing)
